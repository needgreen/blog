## 주요 개념

- 변수 선언 : let, const, var의 차이
- 함수 선언식 / 화살표 함수 : `function` vs `() ⇒ {}`
- **스코프(scope)와 클로저(closure)**
  - 스코프
    - 변수에 접근할 수 있는 범위
    - 전역과 지역(함수 & 블록 내부에서 접근)으로 구분
  - 클로저
    - 클로저는 함수가 선언될 때 스코프를 기억하는 함수
    - 외부 함수의 변수에 접근할 수 있는 내부 함수를 의미
- **this의 개념**
  - 메소드 호출
    - 객체의 메소드 호출 시 this는 해당 객체를 가리킴
  - 생성자 함수
    - new 키워드로 함수 호출 시 this는 새로 생성된 객체를 가리킴
  - call, apply, bind 수동 지정된 객체
  - 화살표 함수
    - 화살표 함수의 this는 자신을 감싸고 있는 외부 스코프의 this를 상
- 배열 메소드 : `map` `filter` `forEach` `reduce`
- 객체 다루기 : 구조 분해 할당, 전개 연산자 (`...`)
- 비동기 처리 : `Promise` `async/await`
- ES6 문법 전반 : `import/export` , 템플릿 리터럴, 기본 매개변수 등

### 브라우저 동작 원리

- DOM (Document Object Model)
- BOM (Browser Object Model)
- 렌더링 과정 (HTML → CSSOM → Render Tree)
- 이벤트 버블링과 캡처링

### 함수형 프로그래밍 사고 방식

- 리액트 → 함수형 컴포넌트 중심 구조
- 순수 함수와 불변성 개념이 중요
  - 순수 함수 : 입력이 같으면 항상 출력이 같다
  - 불변성 : 원본 데이터를 직접 변경하지 않고 새로운 값을 만들어 낸다.

### JSON과 API 기본

- JSON 형식 이해
- REST API 구조 `GET` `POST` `PUT` `DELETE`
- `fetch` / `axios` 로 데이터 가져오기

### 리액트 연결 개념

- JSX → HTML과 JS의 결합된 문법
- Component → 재사용 가능한 UI 단위
- Props & State → 데이터 전달과 관리
- Hook (useState, useEffect) → 함수형 컴포넌트의 생명주기 및 상태 관리
- Router → 페이지 이동 제어
- Context API / Redux → 전역 상태 관리

### 핵심 함수

1. 배열 메소드
   1. `map` : 리스트 렌더링 → 각 요소를 변환해서 **새로운 배열 반환**, 원본과 같은 길이 배열. 반환값 사용
   2. `filter()` : 조건부 필터링 → 조건을 만족하는 요소만 걸러 **새로운 배열을 반환**
   3. `find()` : 특정 요소 찾기 → 배열에서 조건을 만족하는 **“첫 번째 요소”를 반환**
   4. `includes()` : 포함 여부 확인 → 배열에 **특정 값이 존재하는지 여부**를 true / false 로 반환
2. 구조 분해 할당 : 객체나 배열에서 필요한 값만 꺼내서 변수로 저장하는 문법
   1. 배열 구조 분해
   2. 객체 구조 분해
3. 스프레드 연산자
   1. 배열 복사/합치기 `[...aaa, 2, 3];`
   2. 객체 복사/병합 `{...aaa, title:'a'};`
4. 화살표 함수 `()=>{}`
5. 삼항 연산자 `{ ooo > ? '양수' : '음수 or 0'}`
6. 논리 연산자(`&&` `||` )
7. Optional Chaining (`?.`)
8. 템플릿 리터럴 `${ }`
9. 객체 단축 표기법
10. Promise & async/await
